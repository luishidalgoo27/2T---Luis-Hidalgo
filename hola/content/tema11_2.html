DHCP

El protocolo DHCP es un protocolo de configuracion dinámica de host.

El protocolo DHCP es capaz de asignar dinamicamente:

Dirección IP.
Máscara de red.
Puerta enlace.
Servidor DNS.

El protocolo DHCP es capaz de controlar:

Rango de direcciones asignables.
Direcciones libres y ocupadas.
Exclusiones o reservas de direcciones.
Tiempo de asignación.

El protocolo DHCP lo asigna:

Un router.
Un servidor con un SO de red y servicio DHCP instalado.

Hay cuatro tipos de DCHP:

DHCPDISCOVER Busca servidor. Difusión broadcast. 255.255.255.255
DHCPOFFER: Autoriza Conexión
DHCPREQUEST: Solicita IP, Puerta enlace, máscara y Servidor DNS
DHCPack: Envía datos y queda configurado.

Para Configurar un DHCP como cliente en Windows:

Para configurar el DHCO de forma cliente en un sistema operativo en Windows nos tendremos que ir
a las propiedades del adaptador de red -> después le damos a protocolo de internet versión 4 (TCP/IPV4)
y para que funcione el DHCP en nuestro equipo le damos a obtener una dirección de IP automatica

Instalación de un servicio DHCP

Depende de donde queremos hacer la instalación del servico DHCO es de diferente manera:

En tu roouter: Para hacer la instalación del servico DHCP en tu router lo que tienes que hacer es averiguar la puerta de
enlace de tu router y poner https:// y la puerta de enlace de tu router y poner el usuario y la contraseña.
Por ejemplo: https://192.168.24.1

En Cisco paket tracer: En cisco paket tracer es darle al servidor -> después al router
y después a hombe router y tendras que poner el nombre de usuario y su contraseña.

En Windows server: En windows server primero tenemos que instalar el servido DHCP, una vez lo tengas instalado te vas ha
herramientas administrativas y le damos doble clic a DHCP y ya seria elegir que tipo de IP vamos ha utiizar y creamos
un nuevo ambito que es el rango de la IP,
después le pondremos algunas restricciones o reservas a algunas IP y por ultimo un Tiempo de concesión

Analizar trafico con wireshark:

Para analizar el trafico DHCP con wireshark primero tenemos que tener instalado el wireshark
, una vez lo tenemos instalado lo que tenemos que poner udp.port==67 && udp.port==68y ya se veran todo el trafico DHCP.

DNS

El DNS son las siglas de Domain Name System . Es un servicio fundamental en Internet que se encarga de traducir
los nombres de dominio legibles para los humanos a las direcciones IP numéricas que utilizan los dispositivos para comunicarse
entre sí .

El servicio DNS traduce las IP de los host a nombres de dominio más fáciles de recordar para las personas

Los servicios DNS los proporcionan los ISP(Proveedores de Servicos de Internet) O El SO (servidor en entornos corporativos)

¿Dónde configuramos el servicio DNS en windows?

Windows: Propiedades de los adaptadores de la red -> protocolo TCP/IP
Linux: Icono Propiedades de la conexión o linea comandos . nmcli dev show | grep DNS

Para comprobar el Funcionamineto DNS

Windows y Linux con Comando nslookup
El cual lo que hace es comprobar la resolución de nombres directa e inversa

Instalar Servidores DNS

Cisco Paket Tracer Para instalar un Servidor DNS en cisco paket Tracer primero tenemos que abrir el router
,después vamos a la pestaña de configuración, seleccionamos la pestaña de servicios, activamos la casilla
de verificación de Srvidor DNS y le damos clic en Aplicar.

Windows Server: Para instalar un Servidor DNS en windows Server nos tenemos que ir a administrador de servidor, agregar roles y caracteristicas
le damos a todo siguiente y cuando nos deje seleccionar el servicio que queremos instalar seleccionamos el servidor DNS
y ya le damos a todo siguiente e instalar y una vez que se instala si lo queremos configurar/administrar
nos vamos al inicio de windows herramientas administrativas y le damos doble clic a DNS

¿Que son las entradas o registros DNS?

Las entradas o registros DNS son los que establecen la relación entre IP y nombres de Dominio
Las entradas o registros DNS tienen los siguientes Tipos Basicos:

A. Dirección IPv4
AAAA. Dirección IPv6
CNAME. alias de dominio
MX. Correo ELectronico
XT. Informativas

Nombres de Dominio

Los nombres de Dominio es una asocioación de un nombre a IP

Formato

El formato de los nombres de dominio es el noombre separado por puntos
ejemplos: docs.google.es y medac.es.alu.medac.es

Componentes

Los componenetes de los nombres de dominio es: clientes, servidores y zonas de autoridad

Dominios de primer nivel

Territoriales: .es .de .fr
Genericos: .com .net .org .edu
