¿QUÉ ES UNA IP?


- La dirección IP es un conjunto de números que identifica, de manera lógica y jerárquica, a una interfaz en la red de un dispositivo que utilice el protocolo a nivel de red del modelo TCP/IP. Capa 3. Existen dos tipos la IPv4 y la IPv6

PROFUNDIZANDO


- Una red IP se forma asignando direcciones IP y máscaras de red a los equipos que forman parte de la red

- Una IP puede asignarse de forma manual, de forma estática, donde se configura manualmente la dirección IP en cada dispositivo. También puede asignarse de forma automática mediante DHCP, donde un servidor DHCP, asigna dinámicamente direcciones IP a los dispositivos conectados

- Cuando se asigna una IP, se asigna la propia dirección IP del dispositivo, junto con la máscara de red que define qué parte de la dirección IP se refiere a la red y qué parte se refiere al host

- La asignación de direcciones IP puede ser manual por el administrador de la red en asignación estática o automática mediante DHCP, donde un servidor DHCP dentro de la red asigna las direcciones IP dinámicamente

CLASES DE REDES


- Las clases de redes son un sistema de categorización de direcciones IP en función del rango de direcciones disponibles y la cantidad de hosts que pueden alojarse en la red.

EXPLICACIÓN DETALLADA


- Clase A: Para redes grandes, con 8 bits para la dirección de red y hasta 2^24 direcciones de host. Máscara común: 255.0.0.0.

- Clase B: Para redes moderadas, con 16 bits para la dirección de red y 16 bits para direcciones de host. Máscara común: 255.255.0.0.

- Clase C: Para redes pequeñas y LAN, con 24 bits para la dirección de red y 8 bits para direcciones de host (hasta 256 hosts). Máscara común: 255.255.255.0.

COMANDOS DE RED


- En la siguiente tabla vamos a comprender los comandos más importantes

Comando ipconfig

Se usa para asignar direcciones IP a interfaces y configurar parámetros de interfaces manualmente.
Comando ping

Se usa para solucionar problemas de conectividad, disponibilidad y resolución de nombres.
Comando netsat

Genera visualizaciones que muestran el estado de la red y estadísticas de protocolo.
Comando trace

Sirve para trazar la ruta que hace un paquete entrante que viene desde un host o punto de red hasta tu ordenador
Comando route

Manipula rutas para el perfil activo solamente.

LA PUERTA DE ENLACE: GATEWAY


- Sirve para comunicarnos con otras redes. Corresponde a la IP del dispositivo que nos da acceso a otras redes,que es el router. Debe estar en nuestra red

PRIVADA


- Las direcciones IP privadas son direcciones IP reservadas para uso en redes locales (LAN), como hogares, empresas u otras organizaciones. Estas direcciones se utilizan para identificar dispositivos dentro de una red privada y no son accesibles desde Internet en su forma original. Son asignadas por los administradores de redes locales y se utilizan para facilitar la comunicación dentro de la red interna.

PÚBLICA


- Las direcciones IP públicas son asignadas a dispositivos directamente accesibles desde Internet. Son únicas globalmente y gestionadas por organizaciones como IANA. Se utilizan para identificar y comunicarse con servidores web, routers y otros dispositivos en la red pública, facilitando la conectividad global de Internet.
